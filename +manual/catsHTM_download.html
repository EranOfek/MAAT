The HDF5/HTM large catalog format - catalogs download

Description

The HDF5/HTM format designed to store and provide fast access for large astronomical catalogs with >10^6 rows (Soumagnac & Ofek 2018). The format is based on the HDF5 file format and HDF5 file access utilities which are available on many platforms. The catalog format is designed to allow fast access for cone searches in the range of 1 arcsec to about 1 deg. For fast access, the sources are sorted into hierarchical triangular mesh (HTM).

Access to the catalogs is possible using routines in Python (Python catsHTM download) and MATLAB (MATLAB Astronomy and Astrophysics Toolbox download). See also catsHTM documentation and examples.

Download the catalogs

To download the catalogs:

  1. Download http://astro.weizmann.ac.il/catsHTM/list.euler.wget by right clicking on the link and choosing "Download linked file"
  2. Move the dowloaded file to a directory where you would like to have the HDF5 catalogs (e.g. /home/catsHTM/catalogs)
  3. Edit if necessary, according to whether you need all catalogs or just a subset
  4. execute the file from the command line by running the following commands:
  5. chmod a+x list.euler.wget.txt
  6. Execute: ./list.euler.wget.txt
  7. Checksums are available from: http://astro.weizmann.ac.il/catsHTM/list.checksum

List of available catalogs

Currently, the following catalogs are available in this format (alphabetical order):

  1. 2MASS - 2MASS point source catalog
  2. 2MASSxsc - 2MASS extended source catalog
  3. AKARI - AKARI IR catalog
  4. APASS - AAVSO All Sky Photometric Sky Survey (~5.5x10^7 sources)
  5. Cosmos - Sources in the Cosmos field
  6. DECaLS - DECaLS/DR5
  7. FIRST - (~9.5x10^5 sources)
  8. GAIA/DR1 - (~1.1x10^9 sources).
  9. GAIA/DR2 - GAIA-DR2
  10. GALEX - GALAEX/GR6Plus7 (~1.7x10^8 sources).
  11. HSCv2 - HST source catalog version 2
  12. IPHAS/DR2 - IPHAS (H alpha survey of the Galactic plane)
  13. NEDz - NED redshifts
  14. NVSS - (~1.8x10^6 sources)
  15. PS1 - Pan-STARRS (~2.6x10^9 sources; A cleaned version of the PS1 stack catalog; some missing tiles below declination of zero [being corrected])
  16. PTFpc - PTF photometric catalog
  17. ROSATfsc - ROSAT faint source catalog
  18. SDSSDR10 - SDSS/DR10 - Primary sources from SDSS/DR10 (last photometric release)
  19. SDSS/DR14offset - RA/Dec offsets for SDSS primary sources with z<20 mag.
  20. SkyMapper/DR1 - SkyMapper
  21. SpecSDSS/DR14 - SDSS spectroscopic catalog
  22. Spitzer/IRAGgc
  23. Spitzer/SAGE
  24. SWIREz
  25. UCAC4 (~1.1x10^8 sources)
  26. UKIDSS/DR10
  27. USNOB1 (not yet available)
  28. VISTA/Viking/DR3
  29. VST/ATLAS/DR3
  30. VST/KiDS/DR3
  31. WISE (~5.6x10^8 sources)
  32. XMM - 7.3x10^5 sources) 3XMM-DR7 (Rosen et al. 2016; A&A 26, 590)

Note that the catalog base name used in the access program may be different than the catalog name.
